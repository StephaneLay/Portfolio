RazorLight

RazorLight Overview

This project is a classic twin-stick shooter originally developed during a game jam with the theme: 'The more you have, the worse it gets.' My concept involved allowing bullets to bounce off the edges of the screen and obstacles, creating a scenario where you could potentially harm yourself. As you survive, you progressively fire more bullets at once, transforming the game into a bullet hell experience. Additionally, the game features a save data system and a phase mechanism that alters maps, enemy behavior, and music tracks as the player progresses through different phases.
